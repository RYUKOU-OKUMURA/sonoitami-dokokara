# その痛み、どこから？ — 整体師シミュレーション —

整体の「見立て」をゲームとして体験させる、教育性とエンタメ性を両立したWebアプリ企画です。

2026-03-06時点では、実装前の要件定義・技術選定フェーズです。最初の開発目標は、王道シナリオ1本だけで成立するMVPを丁寧に設計することです。

## 企画の核

- プレイヤーは整体師として、問診・評価・施術を選択する
- 痛い場所が原因とは限らない、という整体的な見方をゲーム化する
- 施術テクニックそのものより、仮説と再評価の重要性を伝える
- 教育、エンタメ、世界観発信の3つを同時に狙う
